  • 后台学习(二)——MySql绿色版配置&Navicat for MySQL

    2017-04-20

    1. 下载

    Windows (x86, 64-bit), ZIP Archive
    No thanks, just start my download.

    2. 添加系统环境变量

    变量名： Path    变量值：E:\mysql\mysql-5.7.18-winx64\bin；

    3. 配置my.ini文件

    解压目录下新建my.ini文件，编辑为

    4. 安装服务

    管理员模式下cmd输入mysqld --install MySQL --defaults-file="E:\mysql\mysql-5.7.18-winx64\my.ini"
    显示Service successfully installed

    同理删除服务mysqld --remove MySQL

    5. 启动MySQL服务

    mysqld --initialize自动生成带随机密码的root用户
    mysqld --initialize-insecure自动生成无密码的root用户（我先执行了这个命令）
    data文件夹不为空是不能执行这个命令的。可以先删除data目录下的所有文件或者移走。

    cmd管理员模式在bin目录下net start mysql即可启动服务。
    同理，停止服务器命令net stop MySQL

    • 服务无法启动-“NET HELPMSG 3534”问题解决
      参考http://bbs.csdn.net/topics/391851148
      先mysqld --initialize初始化data目录。
      5.7版本还可以把5.6的data目录拷过去解决。

    • 服务无法启动-1067错误问题解决
      参考http://www.cnblogs.com/mr-hero/p/5817142.html

    6. 设置密码

    mysql -uroot -p password 123456

    7. 本地连接

    cmd输入mysql -u root -p再输密码连接
    退出exit（bye真的好萌(*ﾟ∇ﾟ)）

    8. Navicat for MySQL

    图形化数据库管理工具，官网下载。
    安装后新建连接连接数据库就行。
