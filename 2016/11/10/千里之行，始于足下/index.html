起因

昨晚看到双十一阿里云打折，有几块钱的域名，不如买个建博客。因为个人使用，静态页面基本满足要求，所以没有租虚拟主机（其实想省钱），而是利用了GitHub Pages平台和HEXO搭建。

HEXO是基于Node.js的静态博客程序，相比之下jekyll是基于Ruby的，个人目前不了解这个语言，所以虽然jekyll可以直接原文上传github，而hexo要本地生成后再上传，我还是选择了hexo。

配置环境

Node.js
用命令行检查是否安装好：

node -v
npm -v

Git

git --version
